
KSK LODDSALG-APP – BRUKSVEILEDNING

1. Åpne index.html i en nettleser for å bruke loddsalgs-skjemaet.
2. Loddkjøper fyller ut navn, telefon, antall lodd og velger selger.
3. Test-knappen for Vipps simulerer kjøp – loddnummer tildeles automatisk.
4. Gå til admin.html for administratorvisning.
   - Logg inn med oppgitt brukernavn og passord.
   - Se alle kjøp, rediger selgere, og administrer premier.
   - Trekk vinnere og eksporter data ved behov.
5. Firebase må aktiveres med deres egen prosjektkonfigurasjon.
6. Vipps-integrasjon kan legges til når testingen er godkjent.

Takk for at du bruker systemet!
